# Video Games Sales Analysis
This project explores sales trends for video game sales over years. It demonstrates data cleaning, exploratory analysis, and data visualization in Python. 
I chose video games to do my analysis over because video games have always been a big hobby and part of my life ever since I was very young.  

## Quick Start
Downloading the project to run locally? Here is how you install the required libraries. After you have cloned the repo, open your terminal or cmd navigate to the project folder: 

```python
# windows users
python -m venv venv

# Activating the Virtual Environment for Windows 
source venv/Scripts/activate

# installing libraries from requirements.txt for Windows
pip install -r requirements.txt

# mac / linux
python3 -m venv venv

# Activating the Virtual Environment for Mac / Linux
source venv/bin/activate

# installing libraries from requirements.txt for Mac / Linux
pip3 install -r requirements.txt
```   

## Why am I doing this
My first solo project using EDA and Data Visualizations in Python. The analysis shows:  
    * Trends in Sales over years  
    * Comparison between Genre popularity  
    * Which platforms had the most sales  
    * Comparing ESRB ratings
    * Most Popular games on steam
    * Comparing review scores
